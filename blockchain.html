How to Install

To install the Magicoin Blockchain, replace the "blocks" folder in the following directory based on your operating system:

  • Windows: Replace the "blocks" folder in C:\Users\Your Username Here\AppData\Magi
  • Linux: Replace the "blocks" folder in ~/.magi/
  • macOS: Replace the "blocks" folder in ~/Library/Application Support/Magi/
